# Web and Mobile UI Automation Project
Проект представляет собой комплексное решение для автоматизированного тестирования пользовательских интерфейсов веб-сайта и Android-приложения Wikipedia.
* **Язык разработки:** Java 21
* **Тестовый фреймворк:** TestNG
* **Веб-автоматизация:** Selenium WebDriver
* **Мобильная автоматизация:** Appium (UIAutomator2)
* **Менеджмент драйверов:** WebDriverManager
* **Сборка проекта:** Maven
* **Логирование:** SLF4J + Logback
* **BaseWebTest**: Инициализирует ChromeDriver, настраивает неявные ожидания и максимизирует окно браузера перед каждым тестом.
* **BaseMobileTest**: Настраивает соединение с Appium Server, устанавливает параметры устройства (UDID, имя) и загружает APK-файл приложения.
**Класс WikipediaHomePage (Page Object):**
* `open()`: Открывает главную страницу русской Википедии.
* `enterSearchText(String text)`: Очищает поле поиска, вводит текст и имитирует нажатие клавиши ENTER.
* `clickSearch()`: Выполняет клик по кнопке поиска с обработкой исключений.
* `isMainPageLinkDisplayed()`: Проверяет видимость ссылки на главную страницу.
**Класс WikipediaWebTest:** Содержит 4 сценария: проверка отображения главной страницы, работа поиска, навигация в портал сообщества и проверка языковой панели.
**Класс WikipediaMobilePage (Page Object):**
* `clickSearchContainer()`: Автоматически закрывает приветственные экраны (Onboarding) и нажимает на строку поиска.
* `enterSearchText(String text)`: Вводит поисковый запрос в поле ввода.
* `selectFirstSearchResult()`: Выбирает первый результат из списка с автоматическим закрытием всплывающих уведомлений.
* `getArticleTitle()`: Возвращает текст заголовка открытой статьи.
* `scrollToBottom()`: Выполняет автоматическую прокрутку страницы до самого конца.
**Класс WikipediaMobileTest:** Реализует 3 сценария: базовый поиск, открытие статьи с проверкой заголовка и тестирование корректности скролла контента.
* Установите Java 21 и Maven.
* Для мобильных тестов установите Appium Server (`npm install -g appium`) и драйвер UIAutomator2.
* Положите файл `wikipedia.apk` в корень проекта.
Команда:
mvn clean test -Dtest=WikipediaWebTest
1. Подключите Android устройство через USB и включите отладку по USB.
2. В отдельном терминале запустите сервер: `appium`
3. Запустите тесты командой:
mvn clean test -Dtest=WikipediaMobileTest
Все этапы выполнения тестов записываются в файл `test-output.log` и выводятся в консоль в режиме реального времени для упрощения отладки.
